# Algoritmo de Dijkstra

Proyecto desarrollado en Python utilizando CustomTkinter para encontrar la ruta más corta entre dos nodos mediante el algoritmo de Dijkstra.

## Requisitos

Python 3.10+

Instalar dependencias

pip install -r requirements.txt

## Ejecutar

python main.py
